# Initial Design Assumptions

Date: 2026-05-23

## Spiral Geometry

Use an Archimedean spiral centerline:

```text
r(theta) = r0 + b theta
b = pitch / (2*pi)
x(theta) = r(theta) cos(theta)
y(theta) = r(theta) sin(theta)
```

The channel is generated by offsetting the centerline normal by half the channel width. This gives inner and outer walls without needing a manually drawn CAD file.

## First Device Concept

- One inlet.
- Two or three outlets.
- Side-wall or segmented side-wall electrodes.
- Continuous-flow operation.
- Sorting target: live/dead or viable/nonviable cells.

## First Numerical Outputs

- Outlet assignment for each particle class.
- Separation efficiency.
- Recovery.
- Purity.
- Residence time.
- Minimum voltage at each flow condition.

## Caution

The first prototype should be treated as an analytical design model. Publication-level claims require solved flow, mesh independence, parameter sensitivity, and careful dielectric-property sourcing.
