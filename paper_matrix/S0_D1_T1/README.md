# S0 D1 T1 - No Spiral, DEP, Dead/Live Topic

Status: populated.

These papers use DEP for live/dead or viable/nonviable separation, but not spiral microchannels.

Included examples:

- Elitas et al. 2017: live/dead monocyte separation with 3D carbon electrodes.
- Patel et al. 2012: reservoir-based DEP separation of live/dead yeast cells.
- Huang 2025: ODEP live/dead separation.

Use in project: dielectric parameters, operating frequency ranges, purity/recovery targets, and viability-selective DEP mechanisms.
