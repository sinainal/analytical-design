# S0 D1 T0 - No Spiral, DEP, Not Dead/Live

Status: populated.

These papers use DEP for cell/particle separation or manipulation, but do not focus on spiral geometry or dead/live sorting.

Included examples:

- Nguyen 2024: facing-electrode DEP biological-cell separation.
- Tian 2024: on-chip DEP single-cell manipulation review.

Use in project: electrode strategy, DEP taxonomy, numerical modeling assumptions, and fabrication-oriented design alternatives.
