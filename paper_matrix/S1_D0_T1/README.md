# S1 D0 T1 - Spiral, No DEP, Dead/Live Topic

Status: populated.

These papers use spiral microchannels for dead/nonviable-cell handling, but separation is inertial/passive rather than DEP-driven.

Included example:

- Choi/Kwon 2018: continuous removal of small nonviable mammalian cells and debris from bioreactors using inertial microfluidics.

Use in project: proves the practical relevance of spiral geometry for dead/nonviable-cell handling and gives benchmark metrics for viable-cell retention and dead-cell removal.
